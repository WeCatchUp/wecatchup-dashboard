# WeCatchUp 第四團 KPI Dashboard

## Purpose
Static KPI dashboard for WeCatchUp 第四團 three exercise sub-groups.
Data is updated weekly: download Excel files from JoiiSport → run pipeline → refresh browser.

## File Structure
```
wecatchup-dashboard/
├── CLAUDE.md
├── data/
│   ├── input/               ← place the 6 weekly Excel files here before running update_dashboard.py
│   └── dashboard_data.json  ← generated by update_dashboard.py (overwritten each week)
├── src/
│   └── update_dashboard.py  ← weekly data pipeline (Python 3 + pandas)
└── public/
    └── index.html           ← dashboard UI (open via local HTTP server, NOT file://)
```

## Weekly Update Workflow
1. Download 6 Excel files from JoiiSport (IndividualReport + ExerciseReport per group)
2. Place in `data/input/` using the naming convention below
3. Run the pipeline script:
```bash
python src/update_dashboard.py \
  --run-individual   data/input/run_individual.xlsx \
  --run-exercise     data/input/run_exercise.xlsx \
  --bike-individual  data/input/bike_individual.xlsx \
  --bike-exercise    data/input/bike_exercise.xlsx \
  --walk-individual  data/input/walk_individual.xlsx \
  --walk-exercise    data/input/walk_exercise.xlsx
```
4. Serve the project root with a local HTTP server:
```bash
python -m http.server 8000
```
5. Open http://localhost:8000/public/index.html in browser and refresh

## Three Sub-Groups
| ID   | Name                          | Sport Types               | Weekly Goal | Total Goal |
|------|-------------------------------|---------------------------|-------------|------------|
| run  | RUN CLUB 10週慢跑挑戰賽        | 跑步, 室內跑步             | 20 km       | 200 km     |
| bike | BIKE Training 10週單車挑戰賽   | 自行車, 飛輪               | 40 km       | 400 km     |
| walk | Walk Together 10週健走挑戰賽   | 健行, 超慢跑, 健走杖, 登山 | 15 km       | 150 km     |

## Activity Period
2026-04-27 to 2026-07-05 (10 weeks)

## Tech Stack
- Frontend: Pure HTML + TailwindCSS (CDN) + Chart.js (CDN) — no build step
- Pipeline: Python 3 + pandas (install: `pip install pandas openpyxl`)
- No npm, no React, no framework

## Color Scheme
- Run:  #3b82f6 (blue)
- Bike: #f97316 (orange)
- Walk: #22c55e (green)
- Background: #f2f4f8

## Excel Column Reference
**IndividualReport**: 群組代碼, 群組名稱, 成員編號, 姓名, Email(JoiiSports帳號), 團長, --, 70天個人距離, 日均距離
- `--` == `✅` → 達標; `團長` not null → exclude from 淨 counts

**ExerciseReport**: 群組, 成員編號, 姓名, 性別, 年齡, App顯示名稱, Email(App帳號), 運動日期, 運動類型, 開始時間, 結束時間, 運動時長, 熱量(大卡), 距離(公里), ...
- `運動時長` in seconds (÷60 = minutes); `開始時間` extract hour for time-of-day distribution

## Note on Existing React Dashboard
The `src/`, `public/index.html`, `package.json`, and `build/` in this repo belong to the
legacy Money 第三團 React dashboard. The 第四團 dashboard (`public/index.html`) replaces
the React template for static use. Do not run `npm start` for the 第四團 dashboard.
